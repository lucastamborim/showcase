## 🛠️ Stack

| Camada | Tecnologias |
|---|---|
| **Framework** | Next.js |
| **Frontend** | React · TypeScript |
| **Mobile** | React Native |
| **Backend** | Node.js · Python · Django · DRF |
| **Banco de Dados** | MySQL |
| **DevOps** | Docker · Bitbucket |
| **Deploy** | Vercel |

---

## 📦 Build & Deploy

O projeto é deployado automaticamente na **Vercel** a cada push na branch `main`.

```bash
# Build de produção local
npm run build
npm start
```
